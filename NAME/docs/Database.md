# Database

## Schema (`database/schema.sql`)

Tables: `phcs`, `users`, `stock`, `transfers`, `forecasts`, `alerts`, `feature_snapshots`, `medicine_mappings`

## Seed Data (`database/seed/seed.sql`)

- 4 PHCs in Bangalore Urban/Rural districts
- 7 demo users with various roles
- Starting stock levels with surplus/deficit scenarios
- Feature snapshots for consumption rate simulation
- Medicine alias mappings for fuzzy matching
